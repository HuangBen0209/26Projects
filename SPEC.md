# FaceFit - 脸部评估系统

## 1. 项目概述

**项目名称**: FaceFit  
**类型**: Electron 桌面应用  
**核心功能**: 基于计算机视觉的脸部评估系统，提供面部结构分析、颜值评分和表情情绪识别  
**目标用户**: 普通用户、摄影师、美容顾问

## 3. 功能规范

### 3.1 核心功能模块

#### 3.1.1 面部结构分析
- **人脸检测**: 使用 TensorFlow.js face-landmarks-detection 或 MediaPipe
- **关键点检测**: 468个面部关键点
- **脸型分类**: 椭圆、方脸、圆脸、心形、长脸、国字脸
- **对称性分析**: 左右面部对称程度百分比
- **五官比例**: 三庭五眼分析

#### 3.1.2 颜值评分
- **评分范围**: 0-100分
- **评分维度**: 整体协调性、对称性、成熟度
- **评分可视化**: 环形进度图

#### 3.1.3 表情与情绪识别
- **表情类型**: 高兴、悲伤、惊讶、愤怒、恐惧、厌恶、自然
- **情绪值**: 每种情绪0-100%概率
- **实时追踪**: 支持视频流实时分析

### 3.2 用户交互流程

1. **启动应用** → 显示欢迎页/上次分析
2. **选择输入** → 上传图片 / 打开摄像头
3. **人脸检测** → 检测并框选人脸
4. **执行分析** → 面部结构 + 颜值 + 表情
5. **展示结果** → 分模块展示详细报告
6. **导出报告** → 保存为图片/PDF

### 3.3 数据处理

- **输入**: 图片文件 (JPG, PNG) 或摄像头实时流
- **处理**: 客户端本地处理，不上传服务器
- **输出**: 分析结果本地保存

## 4. 技术架构

### 4.1 技术栈

- **框架**: Electron 28+
- **前端**: React 18 + TypeScript
- **UI**: TailwindCSS
- **计算机视觉**: MediaPipe Face Mesh + TensorFlow.js
- **状态管理**: Zustand
- **构建工具**: Vite + electron-builder
