# FitBlock 健身小游戏 - 技术规格说明书

## 1. 项目概述

### 1.1 项目名称与定位

- **项目名称**: FitBlock（体趣方块）
- **项目类型**: 交互式健身游戏 Web 应用
- **核心定位**: 基于计算机视觉的体感俄罗斯方块游戏，将娱乐与上肢健身结合

### 1.2 核心功能概述

- 经典俄罗斯方块核心玩法（下落、移动、旋转、消除）
- 6种手臂动作体感控制（左右挥臂、双手抬起、双臂下压、单臂前推、双臂交叉）
- 实时健身数据统计（动作次数、卡路里、手臂平衡度）
- 三种健身模式（肩颈放松、燃脂强化、康复训练）

### 1.3 目标用户

- 健身新手：希望通过趣味方式入门运动
- 办公族：久坐肩颈不适，需轻量上肢运动
- 学生群体：追求趣味娱乐结合
- 轻度康复人群：上肢康复训练辅助

## 2. 技术方案

### 2.1 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript（ES6+）
- **姿态识别**: MediaPipe Pose（Google 开源）
- **游戏引擎**: Canvas 2D 自研
- **实时通信**: WebSocket（联机对战）
- **构建工具**: Vite

## 3. 功能详情

### 3.1 经典俄罗斯方块玩法

#### 方块类型
- I型、L型、J型、O型、S型、T型、Z型（共7种）

#### 操作控制
| 操作 | 键盘 | 体感 |
|------|------|------|
| 左移 | ← | 左臂水平左挥（夹角≥60°）|
| 右移 | → | 右臂水平右挥（夹角≥60°）|
| 旋转 | ↑ | 双臂同时抬起（夹角≥90°）|
| 快速下落 | ↓ | 双臂快速下压（≤1s）|
| 暂停1s | - | 单臂前推（夹角≥120°）|
| 暂停/继续 | Space | 双臂交叉 |

#### 得分规则
- 单消: 10分
- 双消: 30分
- 三消: 60分
- 四消: 100分
- 连击加成: 每多1次 +20分

#### 难度递进
- 初始等级: 1级
- 每消除10行: 等级+1
- 最高等级: 10级

### 3.2 姿态识别

#### 识别范围
- 肩关节（左右）
- 肘关节（左右）
- 腕关节（左右）

#### 动作阈值
| 动作 | 判定条件 |
|------|----------|
| 水平左挥 | 左臂与身体夹角 ≥ 60° |
| 水平右挥 | 右臂与身体夹角 ≥ 60° |
| 双臂抬起 | 双臂与身体夹角 ≥ 90° |
| 双臂下压 | 双臂从抬起至放下耗时 ≤ 1s |
| 单臂前推 | 手臂与身体夹角 ≥ 120° |
| 双臂交叉 | 左右腕关节交叉于胸前 |

### 3.3 健身数据统计

#### 统计指标
- 有效动作次数（分左右臂）
- 卡路里消耗估算
- 手臂运动平衡度
- 运动时长

#### 卡路里计算
- 单次挥臂消耗 ≈ 0.5 千卡
- 快速下落（双压）消耗 ≈ 0.3 千卡
- 前推动作消耗 ≈ 0.8 千卡

### 3.4 健身模式

#### 肩颈放松模式
- 方块下落速度降低30%
- 动作阈值降低20%
- 推荐肩颈不适人群

#### 燃脂强化模式
- 方块下落速度提升20%
- 动作阈值标准
- 推荐减脂人群

#### 康复训练模式
- 方块下落速度降低50%
- 动作阈值可自定义
- 支持康复师参数调整
